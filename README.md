# atividade-pratica-programacao-web

Este repositório contém uma página simples construída como exercício de programação web. A estrutura inclui:

- `index.html` com HTML semântico (header, main, sections, footer) e conteúdo organizado conforme o layout fornecido na imagem de design.
- `style.css` com estilos separados, utilizando variáveis CSS, tipografia, cores e espaçamentos definidos para corresponder ao design.
- A imagem de cabeçalho referenciada (`image.png`) utilizada na seção de header.

O layout reproduz fielmente o modelo mostrado no arquivo de design, com destaque em azul para palavras-chave, tipografia clara e responsividade para dispositivos móveis.

As cores, fontes e espaçamentos seguem boas práticas de desenvolvimento e o CSS está organizado por seções para facilitar manutenção.
